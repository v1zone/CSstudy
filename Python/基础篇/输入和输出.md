输出函数print()：以字符形式向控制台输出结果

1  print()用法1：将单一字符串或变量直接输出

```python
a = "人生苦短，我用Python。"
print(a)
```

2  print()用法2：将多个字符串或变量直接输出

```python
a = "人生苦短，"
b = "我用Python。"
print(a, b)
```

3  print()用法3：字符串和变量的混合输出

```python
a = "人生苦短，请用Python。"
print("Python对我说：{}".format(a))
```
